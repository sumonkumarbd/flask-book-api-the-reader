Access the API endpoints for managing PDFs and metadata

API Endpoints for The Reader Android App

Below is the list of available endpoints to interact with The Reader app's backend for PDF management:

READ - Get All PDFs

Fetch a list of all PDFs uploaded in the system.

GET https://flask-book-api-the-reader.onrender.com/pdfs

UPLOAD - Upload PDF

Upload a new PDF along with metadata such as title, author, category, and thumbnail image.

POST https://flask-book-api-the-reader.onrender.com/upload

DELETE - Delete PDF

Delete a PDF and its associated metadata by its ID.

DELETE https://flask-book-api-the-reader.onrender.com/delete/{id}

GET - Get PDF Metadata

Retrieve the metadata of a specific PDF by its ID.

GET https://flask-book-api-the-reader.onrender.com/pdf/{id}

UPDATE - Update PDF Metadata

Update the metadata (title, author, category, etc.) of an existing PDF.

PUT https://flask-book-api-the-reader.onrender.com/update/{id}

SEARCH - Search PDFs

Search for PDFs by keywords in their metadata.

GET https://flask-book-api-the-reader.onrender.com/search?q={query}
