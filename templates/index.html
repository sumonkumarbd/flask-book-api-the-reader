Get All PDFs

Fetches a list of all PDFs uploaded in the system with their associated metadata.

https://flask-book-api-the-reader.onrender.com/api/pdfs

Response Format

[ { "id": 1, "title": "Python Crash Course", "author": "Eric Matthes", "category": "Programming", "thumbnail": "path/to/thumbnail.jpg", "file_name": "python_crash_course.pdf" }, { "id": 2, "title": "Clean Code", "author": "Robert C. Martin", "category": "Programming", "thumbnail": "path/to/thumbnail.jpg", "file_name": "clean_code.pdf" } ]

Sample Code (Android)

// Using Retrofit in Android interface ReaderApiService { @GET("api/pdfs") suspend fun getAllPdfs(): List } // Example usage val pdfs = readerApiService.getAllPdfs()
POST

Upload PDF

Upload a new PDF along with metadata such as title, author, category, and thumbnail image.

https://flask-book-api-the-reader.onrender.com/api/upload

Parameters

Parameter Type Required Description
pdf_file File Yes The PDF file to upload
title String Yes Title of the PDF
author String Yes Author of the PDF
category String Yes Category of the PDF
thumbnail File No Thumbnail image for the PDF

Sample Code (Android)

// Using Retrofit in Android with MultipartBody interface ReaderApiService { @Multipart @POST("api/upload") suspend fun uploadPdf( @Part pdfFile: MultipartBody.Part, @Part("title") title: RequestBody, @Part("author") author: RequestBody, @Part("category") category: RequestBody, @Part thumbnail: MultipartBody.Part? ): Response } // Example usage val pdfFile = File(pdfPath) val pdfRequestBody = pdfFile.asRequestBody("application/pdf".toMediaTypeOrNull()) val pdfPart = MultipartBody.Part.createFormData("pdf_file", pdfFile.name, pdfRequestBody) val titleBody = "Clean Code".toRequestBody("text/plain".toMediaTypeOrNull()) val authorBody = "Robert C. Martin".toRequestBody("text/plain".toMediaTypeOrNull()) val categoryBody = "Programming".toRequestBody("text/plain".toMediaTypeOrNull()) val response = readerApiService.uploadPdf(pdfPart, titleBody, authorBody, categoryBody, thumbnailPart)

PUT

Update PDF Metadata

Updates metadata for an existing PDF file without requiring a new file upload.

https://flask-book-api-the-reader.onrender.com/api/pdfs/{id}

Parameters

Parameter Type Required Description
id Integer Yes The ID of the PDF to update
title String No New title for the PDF
author String No New author for the PDF
category String No New category for the PDF
thumbnail File No New thumbnail image for the PDF

Sample Code (Android)

// Using Retrofit in Android interface ReaderApiService { @Multipart @PUT("api/pdfs/{id}") suspend fun updatePdfMetadata( @Path("id") pdfId: Int, @Part("title") title: RequestBody?, @Part("author") author: RequestBody?, @Part("category") category: RequestBody?, @Part thumbnail: MultipartBody.Part? ): Response } // Example usage val titleBody = "Updated Title".toRequestBody("text/plain".toMediaTypeOrNull()) val authorBody = "Updated Author".toRequestBody("text/plain".toMediaTypeOrNull()) val response = readerApiService.updatePdfMetadata(1, titleBody, authorBody, null, null)
GET

Search PDFs

Search for PDFs based on various criteria like title, author, or category.

https://flask-book-api-the-reader.onrender.com/api/search

Query Parameters

Parameter Type Required Description
q String No General search term across all fields
title String No Filter by title
author String No Filter by author
category String No Filter by category
limit Integer No Number of results to return (default: 20)
offset Integer No Number of results to skip (default: 0)

Response Format

{ "total": 42, "limit": 20, "offset": 0, "results": [ { "id": 1, "title": "Python Crash Course", "author": "Eric Matthes", "category": "Programming", "thumbnail": "path/to/thumbnail.jpg", "file_name": "python_crash_course.pdf" }, { "id": 5, "title": "Learning Python", "author": "Mark Lutz", "category": "Programming", "thumbnail": "path/to/thumbnail.jpg", "file_name": "learning_python.pdf" } ] }

Sample Code (Android)

// Using Retrofit in Android interface ReaderApiService { @GET("api/search") suspend fun searchPdfs( @Query("q") query: String? = null, @Query("title") title: String? = null, @Query("author") author: String? = null, @Query("category") category: String? = null, @Query("limit") limit: Int? = null, @Query("offset") offset: Int? = null ): SearchResponse } // Example usage val searchResults = readerApiService.searchPdfs(category = "Programming", limit = 10)
